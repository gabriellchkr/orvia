# ORVIA Studio — site web premium

Site statique prêt à déployer.

## Mise en ligne simple

1. Acheter un nom de domaine, par exemple `orviastudio.com`.
2. Créer un compte Vercel, Netlify ou Hostinger.
3. Uploader le dossier complet.
4. Connecter le domaine.

## Personnalisation rapide

- Remplacer `contact@orviastudio.com` dans `index.html` par ton vrai email.
- Modifier les prix dans la section `pricing` si besoin.
- Nom de marque déjà intégré : `ORVIA Studio`.

## Fichiers

- `index.html` : structure du site
- `styles.css` : design luxe Apple-like
- `script.js` : animations et navbar
